Automatic short answer grading (ASAG)

This service allows the user to use an Automated Short Answer Grading System.
This system is supposed to be capable of assessing students' answers with a mark approximately near that given manually by the teacher..

Arabic natural language processing (arNLP)

This service allows the user to use several pre-processing techniques
in natural language processing for Arabic language such as:
Tokenization, Diacretization, Stemming, Removing Stop words.
